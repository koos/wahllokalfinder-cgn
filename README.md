wahllokalfinder-cgn
===================
## System Requirements:
- `Ruby 2.3.1`

## Installation:
After cloning the repository, run:  
- ` $ bundle install `
- ` $ rails db:setup `
- ` $ rails import:stations `
- ` $ rails import:addresses `

  **NOTE:** Last 2 tasks may take a while until importing all records to Database.
